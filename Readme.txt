Project structure:

main/java/com.williamhill.sports/ -> Page Object classes, BrowserFactory and Utils.
test/java/com.williamhill.sports/ -> Test classes:
    PlaceBetMobileTest - testng test to verify scenario on mobile chrome driver
    PlaceBetTest - testng test to verify scenario on chrome driver with standard settings
    PlaceBetStepDefinition - cucumber PlaceBet scenario step definition
    TestRunner - runner for cucumber test

test/java/resources/ -> Cucumber feature definitions

reports/ -> report generated by Cucumber


How to run:
Pre-requisites:
To run test download Chrome driver, and save it "C:/tools/" directory.
Use "maven install" to download all dependencies, and Cucumber test should start.
